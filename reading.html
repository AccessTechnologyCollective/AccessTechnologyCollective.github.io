Description

Join us in this session to learn about a range of accessibility features available on iOS devices that can assist with reading and writing including:

  • Text to Speech: This feature converts written text into spoken words, allowing users to have text read aloud to them.
  • Speak Screen: This feature allows users to swipe down from the top of the screen with two fingers to have the content of the screen read aloud.
  • Safari Reader allows you to strip away ads, buttons, and navigation bars, so you can focus on only the content you need.
